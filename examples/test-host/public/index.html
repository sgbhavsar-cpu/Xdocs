Demo host for the Xdocs embeddable control. The host issues a short-lived JWT for the selected role from a mock IdP (/auth/token); the control validates it against the IdP's JWKS. Switch Reader → Editor to see the in-page ✎ Edit button appear — no separate “admin” mode. Append ?ISDEV=1 to the URL to open the control's runtime ⚙ dev-config panel (space/version, theme, and live color overrides).
